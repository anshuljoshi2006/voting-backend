# Voting Backend System

A backend API project built using Node.js, Express.js, MongoDB, and JWT Authentication.

## Features

- User signup and signin
- JWT authentication
- Admin role management
- Candidate management
- Voting system
- Vote count API
- MongoDB database integration

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcrypt
- Postman
